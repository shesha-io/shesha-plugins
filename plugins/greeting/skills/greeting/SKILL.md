# Hello World Skill

This is a simple demonstration skill that shows you how to create and structure a basic Claude Code skill.

## Instructions

When this skill is invoked, follow these steps:

1. Greet the user with a friendly "Hello, World!" message
2. Explain that this is a demonstration skill
3. Provide information about the current date and time
4. Share a fun fact or helpful tip about Claude Code skills

## Example Output

When this skill runs, you should provide output similar to:

```
Hello, World!

This is a demonstration skill from the Shesha plugins marketplace.
Today's date is [current date].

Fun fact: Claude Code skills are activated by natural language!
Just describe what you want to do, and if a skill matches,
Claude will use it automatically.
```

## When to Use This Skill

- User asks to "say hello" or "hello world"
- User wants to test if the plugin is working
- User asks to demonstrate a basic skill
- You want to verify the skill system is functioning
